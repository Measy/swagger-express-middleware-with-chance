Swagger Express Middleware of chancejs mock version
============================
#### Swagger middleware and mocks for Express.js


Fork from [swagger-express-middleware](https://github.com/BigstickCarpet/swagger-express-middleware)

Mock data using [swagger-mock-parser](https://github.com/whq731/swagger-mock-parser)

This Swagger middleware support random mock data while response schema has no example or default defined.

###Example
````yaml
paths
  /api/mis/info:
    get:
      summary: Product Types
      description: |
        query mis
      parameters:
        - name: queryString
          in: query
          required: true
          type: string
          description: mis info
      responses:
        '200':
          description: mis user list
          schema:
            $ref: '#/definitions/MisInfoResult'
definitions:
  MisInfo:
    type: object
    properties:
      id:
        type: integer
        format: int32
        description: key
      login:
        type: string
        description: id
      name:
        type: string
        description: name
  MisInfoResult:
      type: object
      properties:
        code:
          type: integer
          format: int32
          description:
        message:
          type: string
        data:
          type: array
          items:
            $ref: '#/definitions/MisInfo'
````
